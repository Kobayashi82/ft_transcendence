# ft_transcendence
"Single Elimination" o en español "Eliminación Directa" (pares)
Swiss System (Sistema Suizo) (pares e impares)
Todos los jugadores/jugadoras juegan la misma cantidad de rondas.

En cada ronda, se emparejan jugadores con puntuaciones similares.

Nadie queda eliminado.

El ranking final se basa en los puntos acumulados.

Ideal cuando hay muchos participantes y poco tiempo.
